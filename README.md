# 🌐 منصة وصول | Wusool 

**المرجع الرقمي الأول لجاهزية مدارس محافظة العقبة لذوي الإعاقة** 



## 📝 نبذة عن المشروع (About)

مشروع **"وصول"** هو مبادرة تقنية تهدف لتحسين جودة الحياة التعليمية في العقبة عبر توفير قاعدة بيانات لـ 25 مدرسة تدعم الوصول الشامل 



**الرؤية المستقبلية:** نخطط للتوسع ليشمل المشروع كافة محافظات المملكة الأردنية الهاشمية لتكون منصة وطنية شاملة



## 🚀 الميزات والتقنيات (Features & Tech)



*   **نظام البحث:** محرك داخلي للتصفية والوصول للمدارس بسرعة


*   **التصميم:** واجهة متجاوبة (Responsive) تعمل على كافة الشاشات



*   **التقنيات:** تم استخدام HTML5, CSS3, JavaScript, و JSON لإدارة البيانات



*   **الاستضافة:** الموقع مرفوع ومستضاف عبر سيرفرات AWS




## 📂 هيكلية الملفات (File Structure)



*   **`index.html`**: الواجهة الرئيسية ولوحة المؤشرات الإحصائية للمنصة

*   **`schools.html`**: محرك البحث وعرض قائمة المدارس حسب مستوى الجاهزية

*   **`educational-access.html`**: بوابة "وصول التعليمية" والمحتوى التدريبي

*   **`map.html`**: الخريطة التفاعلية لتحديد المواقع الجغرافية للمدارس

*   **`btec.html`**: قسم المدارس التي تدعم المسار المهني الدولي (BTEC)

*   **`favorites.html`**: قائمة المدارس المفضلة التي تم حفظها من قبل المستخدم

*   **`about.html` & `contact.html`**: صفحات التعريف بالرؤية الأكاديمية ووسائل التواصل

*   **`login.html`**: بوابة الوصول وتوثيق الحسابات

*   **`data/`**: المجلد المسؤول عن تخزين البيانات بصيغة JSON (`schools.json`, `users.json`)

*   **`assets/`**: يحتوي على الموارد المساعدة من تصميمات (`css`)، برمجة (`js`)، صور، ومؤثرات صوتية
